# Gimbal

## Gimbal frame

### Bearings
Prepare them as the first step.
You will get three bearings with long bolts and two bearings with short bolts.

**Tighten it thoroughly!**

### Gimbal frame side

Prepare four frame sides: 
- two with longer bolts.
- two with shorter bolts.

### Roll axis frame

Use two frame sides with shorter bolts for roll axis frame.


[<img src="../../assets/images/assembly/gimbal_step_13.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)

Install longer bolts to frame cores as it will not be possible to install them later. Fix one with coupling tube and tighten it, and one with a nut.  
[<img src="../../assets/images/assembly/gimbal_step_16.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)
[<img src="../../assets/images/assembly/gimbal_step_17.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)

Start assembly from frame side:
 - Install arm lever on it and tighten M8 nut fixing it.
 - Install frame cores into corresponding slots.
 - Install second frame side, and tighten M8 nut fixing it.
 
**Note how frame sides oriented to the mowing gimbal arm!**

[<img src="../../assets/images/assembly/gimbal_step_22.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)
[<img src="../../assets/images/assembly/gimbal_step_23.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)
[<img src="../../assets/images/assembly/gimbal_step_24.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)
[<img src="../../assets/images/assembly/gimbal_step_25.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)
[<img src="../../assets/images/assembly/gimbal_step_27.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)
[<img src="../../assets/images/assembly/gimbal_step_28.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)

Fix gimbal sides to gimbal cores using nut in slot and tightening screws.

[<img src="../../assets/images/assembly/gimbal_step_30.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)
[<img src="../../assets/images/assembly/gimbal_step_31.jpg" width="360">](../../assets/images/assembly/gimbal_step_01.jpg)

Install pulley onto the frame:
 - unscrew temporary M8 nut.
 - install spacers 6mm in total.
 - install pulley.
 - fix with M6 bolts and M8 nut.
